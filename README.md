# ABS-automation

경희대학교 중앙동아리 소속 주식경제동아리 ABS의 **동아리방 야간 사용신청서 자동화 시스템**입니다.

본 프로젝트는 반복적으로 작성해야 하는 동아리방 야간 사용신청서를 자동으로 생성하고,<br> 지정된 수신처로 이메일 발송까지 처리하기 위해 제작되었습니다.

---

## 1. 프로젝트 개요

**ABS-automation**은 ABS 운영진의 행정 업무를 줄이기 위한 자동화 도구입니다.

기존에는 동아리방 야간 사용신청서를 수동으로 작성하고 이메일로 제출해야 했지만, <br>본 프로젝트를 활용하면 정해진 양식에 맞춰 문서를 자동 생성하고 이메일 발송 과정을 자동화할 수 있습니다.

- **소속:** 경희대학교 중앙동아리 소속 주식경제동아리 ABS
- **목적:** 동아리방 야간 사용신청서 작성 및 발송 자동화
- **기능:** 신청서 문서 생성, 이메일 자동 발송, GitHub Actions 기반 자동 실행
- **제작:** ABS 37기 최석훈
- **완성:** 2026.05.07

---

## 2. 주요 기능

- **신청서 자동 생성:** `template.docx` 양식을 기반으로 동아리방 야간 사용신청서를 생성합니다.
- **이메일 자동 발송:** 생성된 신청서를 지정된 수신자에게 이메일로 발송합니다.
- **GitHub Actions 연동:** `.github/workflows` 설정을 통해 정해진 조건에 따라 자동 실행할 수 있습니다.
- **반복 업무 절감:** 매번 동일한 양식을 직접 수정하고 발송하는 과정을 줄일 수 있습니다.
